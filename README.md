# ShibauraBBS

## Overview
Internet discussion board using WebSocket

※Only students of Shibaura Institute of Technology

## How to work
### Packages
- Flask
- Flask-SocketIO
- PyJWT
- cryptography

### Install
```
pip install flask flask-socketio pyjwt cryptography
```

###  Run
# DBの初期化
```
python createDB.py
```
# サーバの場合
```
python server.py
```
or
# ローカル環境の場合
```
python app.py
```
